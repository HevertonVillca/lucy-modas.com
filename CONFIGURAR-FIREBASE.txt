LUCY MODAS — CONFIGURAR FIREBASE (5 minutos)
==========================================

1. Acesse https://console.firebase.google.com
2. Criar projeto "Lucy Modas"
3. Firestore Database → Criar banco → modo produção
4. Regras → colar conteúdo do arquivo firestore.rules → Publicar
5. Configurações do projeto → Seus apps → Web (</>) 
6. Copiar firebaseConfig e colar em firebase-config.js (substituir COLE_SUA_API_KEY etc.)
7. Hospedar o site (Firebase Hosting, Netlify ou Live Server) — não abrir só como arquivo file://
